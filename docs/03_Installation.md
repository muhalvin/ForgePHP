# Installation

1. Clone the repository:

```bash
git clone https://github.com/muhalvin/forgephp.git
cd forgephp
```

2. Install dependencies:

```bash
composer install
```

3. Set up your `.env` for database credentials.

4. Run the built-in PHP server:

```bash
php -S localhost:8000
```

5. Use the CLI tool:

```bash
php forge
```
